    📖 Comment utiliser Lyric Video Pro

    1

    Créez un compte

    Cliquez "Lancer l'application", inscrivez-vous avec votre email. Le plan gratuit inclut les fonctions de base.

    2

    Nouveau projet + Audio

    Cliquez "Nouveau projet", entrez titre et artiste. Glissez votre fichier audio (MP3, WAV, AAC) dans la zone audio.

    3

    Collez vos paroles

    Dans PAROLES, collez tout le texte de votre chanson. Choisissez le mode de séparation et cliquez "Analyser & Séparer".

    4

    Synchronisez BPM + Offset

    BPM : Tapez "🥁 TAPER AU RYTHME" plusieurs fois — le BPM est détecté automatiquement.
    Offset : Cliquez "⏱ TAP START" quand les paroles commencent — toutes les lignes se repositionnent.

    5

    Éditez la Timeline

    Glissez les blocs pour déplacer le timing. Redimensionnez le bord droit pour changer la durée. Utilisez + ZOOM / − ZOOM. Cliquez MARK pendant la lecture pour enregistrer.

    6

    Personnalisez le style

    Police, animation, couleurs, fond du texte et arrière-plan dans le panneau gauche. Les éléments 🔒 nécessitent le plan PRO.

    7

    Mode Grand Écran

    Appuyez F ou cliquez "⛶ Plein écran" pour projeter les paroles — parfait pour les cultes et concerts. Flèches ◀▶ pour avancer. Échap pour quitter.

    8

    ⭐ PRO — Exportez en MP4

    Avec le plan PRO ($15.78/mois), cliquez "📤 Exporter". Choisissez HD, Full HD ou 4K. La vidéo est prête pour YouTube, Instagram ou TikTok.

    💡 Raccourcis clavier
    Espace Play / Pause → Ligne suivante ← Ligne précédente F Plein écran Ctrl+S Sauvegarder Échap Quitter plein écran
